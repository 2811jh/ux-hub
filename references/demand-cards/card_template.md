# 需求卡片格式模板

## 单条卡片

```md
## 需求 {序号}
- **需求名称**：{项目名称}
- **需求描述**：{结构化描述}
```

## 字段说明

| 字段 | 说明 | 来源 |
|------|------|------|
| 需求名称 | 提炼后的项目名称，简洁准确 | 从原文提取核心项目名 |
| 需求描述 | 结构化整理后的完整描述 | 从原文提炼背景、目标、重点 |

## 整理原则

1. 需求名称尽量保留原始项目命名，清理噪声字符
2. 需求描述聚焦调研目标、关注重点、产出物
3. 不凭空添加原文未提及的信息
4. 描述控制在 1～3 句，简洁专业
